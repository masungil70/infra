### 정적 IP 설정하는 방법 

variables.tf 파일에 정적 IP 설정을 위한 변수를 선언한다 
```
vi variables.tf
...
variable "static_ip" {
  type        = string
  description = "VM에 할당할 정적 IP 주소"
}

variable "static_ip_cidr" {
  type        = number
  description = "정적 IP의 CIDR 접두사 (예: 24)"
}

variable "static_gateway" {
  type        = string
  description = "VM의 기본 게이트웨이 주소"
}
```

terraform.tfvars 파일에 정적 IP 설정을 위한 변수에 값을 대입한다 
```
vi terraform.tfvars
...

# 네트워크 정보
static_ip = "192.168.0.4"
static_ip_cidr = 24
static_gateway = "192.168.0.1"

```

step8.tf 파일에 DHCP를 정적 IP로 설정을 변경합니다 
```
...
  # Cloud-Init을 통한 IP 설정 (DHCP로 IP 자동 할당)
  # ipconfig0 = "ip=dhcp" 이부분을 아래와 같이 변경한다 

  # Cloud-Init을 통한 IP 설정 (정적 IP 할당)
  ipconfig0 = "ip=${var.static_ip}/${var.static_ip_cidr},gw=${var.static_gateway}"

...
```

terraform init
terraform plan
