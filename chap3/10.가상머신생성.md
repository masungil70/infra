# 가상머신 생성시 Network VLAN 설정

- 가상머신 이름 : server_temp
- ISO 이미지 : ubuntu-24.04.xxx
- CPU : 2 core
- RAM : 2048M
- 네트워크
    - 브릿지 : bmbr0
    - 모델 : VirIO(반가상화)
    - VLAN 태그 : 10
    - MAC 주소 : auto

이 상태로 가상머신을 생성합니다.

설치시 DHCP로 pfSense로 부터 받은 아이피를 확인합니다.
   172.16.10.100 ~ 172.16.10.199 사이의 아이피 대역을 확인합니다

설치가 완료된 후 ip a로 주소를 다시 확인합니다.

sudo apt update가 정상 진행되는 확인합니다

sudo apt upgrade가 정상 진행되는 확인합니다

sudo apt install iputils-ping가 정상 설치가 되는지 확인합니다

설치 후 ping 8.8.8.8이 되는지 확인을 합니다.

새로운 가상 머신 VLAN 태그를 20,30,40으로 해서 생성합니다

현재 각 VLAN의 IP 대역에서는 방화벽 목적지를 any 해서 모두 정상으로 ping 되는 것을 확인 할 수 있습니다

업무 보안 정책에 맞게 네트워크 보안 정책을 방화벽에서 설정합니다.
