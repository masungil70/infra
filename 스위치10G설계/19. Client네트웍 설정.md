# 19. Client네트웍 설정

## Client 1 ~ 16까지 설정 

1. 기존 설정 저장

```bash
cp /etc/network/interfaces /etc/network/interfaces.org
```

---

2. /etc/network/interfaces 파일 수정

```bash
vi /etc/network/interfaces


auto lo
iface lo inet loopback

#################################
# 1. 관리망 (1G NIC)
#################################
iface eno1 inet manual

auto vmbr0
iface vmbr0 inet static
    # 각 Client 별로 관리형 ip를 설정합나다
    address 172.16.53.110/24
    gateway 172.16.53.1
    bridge-ports eno1
    bridge-stp off
    bridge-fd 0

#################################
# 2. 10G NIC
#################################
iface enp1s0f0 inet manual
    mtu 9000

iface enp1s0f1 inet manual
    mtu 9000

#################################
# 3. Bond
#################################
auto bond0
iface bond0 inet manual
    bond-slaves enp1s0f0 enp1s0f1
    bond-mode active-backup
    bond-miimon 100
    mtu 9000

#################################
# 4. Ceph 전용 Bridge (TRUNK)
#################################
auto vmbr1
iface vmbr1 inet manual
    bridge-ports bond0
    bridge-stp off
    bridge-fd 0
    bridge-vids 10 20
    mtu 9000

#################################
# 5. VLAN 인터페이스
#################################
auto vmbr1.10
iface vmbr1.10 inet static
    address 10.10.10.13/24
    mtu 9000

auto vmbr1.20
iface vmbr1.20 inet static
    address 10.10.20.13/24
    mtu 9000
	
	
```

3. 네트웍 설정 적용 

```bash
ifreload -a 

```

4. 네트웍 설정 확인 

```bash
ip a


```

- 설정된 관리용 ip 를 확인합니다 
- 설정된 vlan 10의 ip를 확인합니다
- 설정된 vlan 20의 ip를 확인합니다

## Ceph Node4(Ceph-C4) 설정 

1. 기존 설정 저장 

```
cp /etc/network/interfaces /etc/network/interfaces.org
```

---

/etc/network/interfaces 파일 수정 

```
vi /etc/network/interfaces


auto lo
iface lo inet loopback

#################################
# 1. 관리망 (1G NIC)
#################################
iface eno1 inet manual

auto vmbr0
iface vmbr0 inet static
    address 172.16.53.110/24
    gateway 172.16.53.1
    bridge-ports eno1
    bridge-stp off
    bridge-fd 0

#################################
# 2. 10G NIC
#################################
iface enp1s0f0 inet manual
    mtu 9000

iface enp1s0f1 inet manual
    mtu 9000

#################################
# 3. Bond
#################################
auto bond0
iface bond0 inet manual
    bond-slaves enp1s0f0 enp1s0f1
    bond-mode active-backup
    bond-miimon 100
    mtu 9000

#################################
# 4. Ceph 전용 Bridge (TRUNK)
#################################
auto vmbr1
iface vmbr1 inet manual
    bridge-ports bond0
    bridge-stp off
    bridge-fd 0
    bridge-vlan-aware yes
    bridge-vids 10 20
    mtu 9000

#################################
# 5. VLAN 인터페이스
#################################
auto vmbr1.10
iface vmbr1.10 inet static
    address 10.10.10.14/24
    mtu 9000

auto vmbr1.20
iface vmbr1.20 inet static
    address 10.10.20.14/24
    mtu 9000
	
	
```

3. 네트웍 설정 적용 

```bash
ifreload -a 

```

4. 네트웍 설정 확인 

```bash
ip a


```

- 설정된 관리용 ip 를 확인합니다 
- 설정된 vlan 10의 ip를 확인합니다
- 설정된 vlan 20의 ip를 확인합니다
