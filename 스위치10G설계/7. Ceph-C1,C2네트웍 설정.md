# 7. 네트웍 설정

## Ceph Node1(Ceph-C1) 설정

### 1. 기존 설정 저장

```bash
cp /etc/network/interfaces /etc/network/interfaces.org
```

---

### 2. /etc/network/interfaces 파일 수정

```bash
vi /etc/network/interfaces


auto lo
iface lo inet loopback

#################################
# 1. 관리망 (1G NIC)
#################################
iface eno1 inet manual

auto vmbr0
iface vmbr0 inet static
    address 172.16.53.110/24
    gateway 172.16.53.1
    bridge-ports eno1
    bridge-stp off
    bridge-fd 0

#################################
# 2. 10G NIC (VLAN 10)
#################################
iface enp1s0f0 inet manual
    mtu 9000

auto vmbr1
iface vmbr1 inet manual
    address 10.10.10.31/24
    bridge-ports enp1s0f0
    bridge-stp off
    bridge-fd 0
    mtu 9000
	
```

### 3. 네트웍 설정 적용

```bash
ifreload -a 

```

### 4. 네트웍 설정 확인

```bash
ip a


```

- 설정된 관리용 ip 를 확인합니다
- 설정된 vlan 10의 ip를 확인합니다
- 설정된 vlan 20의 ip를 확인합니다
