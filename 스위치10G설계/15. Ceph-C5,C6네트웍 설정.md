# 15. Ceph-C5, C6네트웍 설정

## Ceph Node5(Ceph-C5) 설정

1. 기존 설정 저장

```bash
cp /etc/network/interfaces /etc/network/interfaces.org
```

---

2. /etc/network/interfaces 파일 수정

```bash
vi /etc/network/interfaces


auto lo
iface lo inet loopback

iface nic0 inet manual

auto vmbr0
iface vmbr0 inet static
        address 172.16.30.11/24
        gateway 172.16.30.1
        bridge-ports nic0
        bridge-stp off 
        bridge-fd 0
        bridge-vlan-aware yes
        bridge-vids 2-4094

#2. 10G NIC
iface nic1 inet manual
        mtu 9000

auto vmbr1
iface vmbr1 inet static
        address 10.10.10.39/24
        bridge-ports nic1
        bridge-stp off
        bridge-fd 0
        mtu 9000

iface nic2 inet manual


source /etc/network/interfaces.d/*
	
	
```

3. 네트웍 설정 적용

```bash
ifreload -a 

```

4. 네트웍 설정 확인

```bash
ip a


```

- 설정된 관리용 ip 를 확인합니다 
- 설정된 vlan 10의 ip를 확인합니다
