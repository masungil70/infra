# 5. Leaf-C1 설정 (Ceph Node 1~2)

```bash
conf t
hostname Leaf-C1
```

---

## Spine uplink 설정

```bash
interface Ethernet1/0/1
 switchport mode trunk
 switchport trunk allowed vlan 10;20


interface Ethernet1/0/2
 switchport mode trunk
 switchport trunk allowed vlan 10;20


```
