# 16. spine1 spine2, leaf1, Ceph-C3,C4 서버 설정 검증 

spine1 spine2, leaf1을 설정한 상태에서 Ceph 서버 2(C3,C4)대를 사용하여 네트웍에 문제가 검증합니다.


🔥 VLAN 10 / VLAN 20 / MTU 9000 / Spine-Leaf 구조가 정상인지 검증


## 🧭 전체 테스트 흐름

```
1. 물리 링크 확인
2. VLAN 인터페이스 구성
3. L2 통신 테스트 (ping)
4. MTU 테스트 (jumbo)
5. 대역폭 테스트 (iperf)
6. 장애 테스트 (failover)
```

---

# 1️. 물리 링크 확인

Ceph Node1에서 진행합니다 


## ✔ 네트웍 설정 확인 

```bash
ip a

```

- 설정된 관리용 ip 를 확인합니다 
- 설정된 vlan 10의 ip를 확인합니다
- 설정된 vlan 20의 ip를 확인합니다

---

# 2️. L2 / VLAN 통신 테스트

## ✔ VLAN 10 테스트

Node1:

```bash
ping 10.10.10.11
```

👉 정상:

```
64 bytes from 10.10.10.12: icmp_seq=1 ttl=64 time=0.3 ms
```

---

## ✔ VLAN 20 테스트

```bash
ping 10.10.20.12
```

---

## ❗ 실패 시 체크

* VLAN tagging 문제
* 스위치 trunk 설정 오류
* native VLAN 문제

---

# 3️. MTU 9000 테스트

## ✔ jumbo frame 테스트

Node1:

```bash
ping -M do -s 8972 10.10.10.11
```

👉 정상:

```
no fragmentation
```

---

## ❗ 실패 시 의미

| 증상                 | 원인            |
| -------------------- | ----------      |
| packet fragmentation | MTU 불일치      |
| ping 실패            | 스위치 MTU 문제 |

---

# 4️. 대역폭 테스트 (iperf)

## ✔ Node2 (서버)

```bash
iperf3 -s
```

---

## ✔ Node1 (클라이언트)

```bash
iperf3 -c 10.10.10.11
```

---

## 🎯 기대값

| 환경  | 속도     |
| ---   | -------- |
| 10G   | 8~9 Gbps |
| 1G    | 900 Mbps |

---

# 5️. VLAN 분리 검증 (중요)

👉 일부러 틀리게 테스트

```bash
ping 10.10.10.12  (vmbr1.10에서)
```

👉 결과:

> ❌ 실패해야 정상

---

# 6️. 장애 테스트 (🔥 핵심)

## ✔ 테스트 방법

👉 Leaf에서 uplink 하나 shutdown

```bash
interface Ethernet1/0/1
 shutdown
```

---

## ✔ 기대 결과

* ping 끊김 없음
* STP failover 발생

---

## ✔ 다시 복구

```bash
no shutdown
```

---

# 7️. 스위치 측 확인

## Spine / Leaf에서

```bash
show spanning-tree
```

👉 확인:

* 하나 FWD
* 하나 BLK

---

```bash
show mac-address-table
```

👉 MAC 학습 확인

---

# 8️. 최종 체크리스트

| 항목       | 정상 조건     |
| --------   | ------------- |
| VLAN 통신  | ping OK       |
| MTU 9000   | jumbo ping OK |
| 대역폭     | 정상 속도     |
| VLAN 분리  | 교차 ping 실패|
| STP        | 1 FWD / 1 BLK |
| Failover   | 끊김 없음     |

---

# 🎯 문제 유형별 빠른 판단

| 증상         | 원인            |
| -------      | -------------   |
| ping 안됨    | VLAN mismatch   |
| MTU 실패     | 스위치 MTU      |
| 속도 낮음    | LACP/STP        |
| 간헐적 끊김  | loop / STP 문제 |
