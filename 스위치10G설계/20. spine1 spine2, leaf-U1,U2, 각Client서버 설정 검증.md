# 12. spine1 spine2, leaf-u1, leaf-u2에 연결된 Client1~16 서버 설정 검증

spine1 spine2, leaf-u1, leaf-u2에 연결된 Client1~16 서버를 사용하여 네트웍에 문제가 검증합니다.


🔥 VLAN 10 / VLAN 20 / MTU 9000 / Spine-Leaf 구조가 정상인지 검증


## 🧭 전체 테스트 흐름

```
1. 물리 링크 확인
2. VLAN 인터페이스 구성
3. L2 통신 테스트 (ping)
4. MTU 테스트 (jumbo)
5. 대역폭 테스트 (iperf)
6. 장애 테스트 (failover)
```

---

# 1️. 물리 링크 확인

Client1에서 진행합니다


## ✔ 네트웍 설정 확인 

```bash
ip a

```

- 설정된 관리용 ip 를 확인합니다
- 설정된 vlan 10의 ip를 확인합니다

# 3️. MTU 9000 테스트

## ✔ jumbo frame 테스트

Node1:

```bash
ping -M do -s 8972 10.10.10.11
ping -M do -s 8972 10.10.10.12
ping -M do -s 8972 10.10.10.13
ping -M do -s 8972 10.10.10.14
ping -M do -s 8972 10.10.10.15
ping -M do -s 8972 10.10.10.16
```

👉 정상:

```
no fragmentation
```

---

## ❗ 실패 시 의미

| 증상                 | 원인            |
| -------------------- | ----------      |
| packet fragmentation | MTU 불일치      |
| ping 실패            | 스위치 MTU 문제 |

# 4️. 대역폭 테스트 (iperf)

## ✔ Ceph1 ~ Ceph6 (서버)

```bash
iperf3 -s
```

---

## ✔ Client1 (클라이언트)

```bash
iperf3 -c 10.10.10.11
iperf3 -c 10.10.10.12
iperf3 -c 10.10.10.13
iperf3 -c 10.10.10.14
iperf3 -c 10.10.10.15
iperf3 -c 10.10.10.16
```

---

## 🎯 기대값

| 환경  | 속도     |
| ---   | -------- |
| 10G   | 8~9 Gbps |
| 1G    | 900 Mbps |

# 5️. VLAN 분리 검증 (중요)

👉 일부러 틀리게 테스트

```bash
# Client1에서 (10.10.10.31) ↔ Ceph1 (10.10.10.11)
# Client1에서 실행 
# (정상)
ping 10.10.10.11  
```

## 인터페이스 지정 ping

```bash
# Client1에서 (10.10.10.31) ↔ Ceph1 (10.10.10.11)
# Client1에서 실행 
# (정상)
ping -I vmbr1.10 10.10.10.11
```

-I 나간는 소스 인터페이스지정 옵션

## 실패

```bash
# Client1 (10.10.10.11) ↔ Chep1 (10.10.10.11)
# Client1에서 실행 

# (실패)
ping -I vmbr1 10.10.10.11

# (실패)
ping -I vmbr1.30 10.10.10.11

```

# 6️. 장애 테스트 (🔥 핵심)

## ✔ 테스트 방법

👉 Client가 연결된 스위치에서 uplink 하나 shutdown

```bash
interface Ethernet1/0/1
 shutdown
```

---

## ✔ 기대 결과

* ping 끊김 없음
* STP failover 발생

---

## ✔ 다시 복구

```bash
no shutdown
```

# 8️. 최종 체크리스트

| 항목       | 정상 조건     |
| --------   | ------------- |
| VLAN 통신  | ping OK       |

# 🎯 문제 유형별 빠른 판단

| 증상         | 원인            |
| -------      | -------------   |
| ping 안됨    | VLAN mismatch   |
| MTU 실패     | 스위치 MTU      |
| 속도 낮음    | LACP/STP        |
| 간헐적 끊김  | loop / STP 문제 |
