# 18. Leaf-U1, U2 설정확인

1. 인터페이스 상태 확인 (uplink 살아있는지)

Leaf-U1, Leaf-U2 스위치에 각각 연결 하고 아래 설정을 진행합니다

```
show interface Ethernet1/0/1

```

...

3. trunk 설정 확인 (핵심)

``` 
show running-config interface Ethernet1/0/1
```

결과 
```
!
Interface Ethernet1/0/1
 switchport mode trunk
 switchport trunk allowed vlan 10;20
!

```

```
show running-config interface Ethernet1/0/2
```

결과 
```
!
Interface Ethernet1/0/2
 switchport mode trunk
 switchport trunk allowed vlan 10;20
!

```
