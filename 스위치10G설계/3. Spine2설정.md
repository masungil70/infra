# 3. Spine2 설정 (Spine1과 대칭)

```bash
conf t
hostname Spine2
```

## Spine LACP

```bash
interface Ethernet1/0/1
 port-group 1 mode active

interface Ethernet1/0/2
 port-group 1 mode active

interface Port-Channel1
 switchport mode trunk
 switchport trunk allowed vlan 10;20

```

---
