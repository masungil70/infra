## 2단계: 가상머신(VM) 생성 및 OS 설치

  이제 웹 UI에서 우리가 계획한 사양대로 3개의 VM을 생성합니다.

  1. Xubuntu Desktop VM 생성 (메인 PC 역할)
   * VM 생성 버튼을 클릭하고 아래 사양으로 생성합니다.
       * OS: Xubuntu ISO 이미지 선택
       * 디스크: 스토리지 local-lvm, 크기 64 GB
       * CPU: 코어 2
       * 메모리: 4096 MB (4 GB)
   * 생성 후, VM을 시작하고 웹 UI의 콘솔을 통해 Xubuntu 설치를 진행합니다.

  2. Alpine Linux VM 생성 (경량 서버 1)
   * VM 생성 버튼을 클릭하고 아래 사양으로 생성합니다.
       * OS: Alpine Linux ISO 이미지 선택
       * 디스크: 스토리지 local-lvm, 크기 20 GB
       * CPU: 코어 1
       * 메모리: 1024 MB (1 GB)
   * 생성 후, VM을 시작하고 콘솔을 통해 Alpine Linux 설치를 진행합니다.
   * 콘솔에서 setup-alpine 명령어 입력 하고 실행합니다 

  3. Ubuntu Server VM 생성 (경량 서버 2)
   * VM 생성 버튼을 클릭하고 아래 사양으로 생성합니다.
       * OS: Ubuntu Server ISO 이미지 선택
       * 디스크: 스토리지 local-lvm, 크기 20 GB
       * CPU: 코어 1
       * 메모리: 1024 MB (1 GB)
   * 생성 후, VM을 시작하고 콘솔을 통해 Ubuntu Server 설치를 진행합니다.
