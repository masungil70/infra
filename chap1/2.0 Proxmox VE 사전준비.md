## 0단계: Proxmox VE 사전 준비

 컴퓨터 1대에 Proxmox를 설치하고 그 위에 Xubuntu, Alpine Linux, Ubuntu Server 가상머신을 올려 우리가 그린 시스템 구조를 완성하는 방법을 단계별로 자세히 안내해 드리겠습니다.

  사전 준비

  설치를 시작하기 전에 아래 항목들을 미리 준비합니다.

   1. 하드웨어:
       * 설치를 진행할 컴퓨터 본체 1대
       * 모니터, 키보드, 마우스
       * 인터넷 공유기에 연결된 유선 랜선

   2. 소프트웨어 (미리 다운로드):
       * Proxmox VE 설치 ISO: Proxmox 공식 홈페이지 (https://www.proxmox.com/en/downloads)에서 다운로드합니다.
       * Xubuntu Desktop ISO: xubuntu.org/download 에서 최신 LTS 버전으로 다운로드합니다.
       * Alpine Linux ISO: alpinelinux.org/downloads 에서 'STANDARD', 'x86_64' 버전의 'iso' 파일을 다운로드합니다.
       * Ubuntu Server ISO: ubuntu.com/download/server 에서 최신 LTS 버전으로 다운로드합니다.
       * USB 부팅 디스크 제작 도구: Rufus 또는 BalenaEtcher.

   3. 임시 장비:
       * 초기 설정을 위한 스마트폰, 태블릿, 또는 다른 노트북 (Proxmox 서버와 같은 공유기 Wi-Fi에 연결 가능해야 함)
