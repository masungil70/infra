## 8단계: Ubuntu Server VM 생성

Ubuntu Server VM 시스템 구성 

  * 이름 : ubuntu-server
  * OS: Ubuntu Server ISO 이미지 선택
  * 디스크: 스토리지 local-lvm, 크기 20 GB
  * CPU: 코어 1
  * 메모리: 1024 MB (1 GB)

1. VM 생성 버튼을 클릭하고 아래 사양으로 생성합니다.
2. 일반 탭 화면에서 VM **이름**을 ubuntu-server 입력하고 **부팅 시 시작**을 선택 합니다 
![alt text](image-67.png)

3. OS 탭 화면에서 **OS**: ubuntu-*-server ISO 이미지 선택합니다 
![alt text](image-68.png)

4. 시스템 탭 화면에서 **Qemu 에이전트**을 체크합니다
![alt text](image-7.png)

5. 디스크 탭 화면에서 **스토리지** : local-lvm, **디스크 크기** : 20G로 설정합니다
![alt text](image-35.png)

6. CPU 탭 화면에서 **유형** : host, **코어** : 1 설정합니다
![alt text](image-37.png)

7. 메모리 탭 화면에서 **메모리** : 1024 설정합니다
![alt text](image-38.png)

8. 네트워크 탭 화면에서 기본설정을 사용합니다
![alt text](image-11.png)

9. 확인 탭 화면은 설정된 것을 확인하고 문제가 없으면 [종결]을 클릭합니다
![alt text](image-69.png)

10. 생성된 가상머신을 확인하고 콘솔 버튼을 클릭하고 시작 버튼을 클릭하여 시작합니다 
![alt text](image-70.png)

11. 잠시후 ubuntu 리눅스 설치 화면이 출력되면 엔터를 누르고 다음으로 계속 진행합니다 
![alt text](image-71.png)

12. 언어를 선택하는 화면이 출력되면 언어는 English를 선택하고 [Done] 버튼을 엔터로 눌러 다음으로 진행합니다 .
항목들 사이를 이동을 할 때는 방향키(업, 다운)를 이용하여 이동합니다 
![alt text](image-73.png)

13. 키보드 레이아웃은 Korean, 키보드 유형은 Korean - Korean (101/104-key compatible)을 선택하고 [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
항목의 값이 여러개인 중에서 한개 선택 할 경우 우측에 ▼ 이 있으면 해당 위치로 이동한 후 엔터키를 여러값이 목록으로 나열됩니다. 이때 원하는 값을 방향키로 이동하여 엔터키로 선택하시면 됩니다.
![alt text](image-74.png)

14. 설치 유형은 Ubuntu Server (minimized) 을 선택합니다. 선택은 엔터키 또는 스페이스바를 누르시면 선택됩니다. [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-75.png)


15. 네트워크 연결하기 인터페이스 기본명 [ens18]이 출력되는데, DHCPv4 옆에 할당받은 ip가 출력됩니다. 할당된 ip 확인하고 [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-76.png)


16. 프록시 서버 설정은 기본으로 하고  [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-77.png)

17. Ubuntu archive mirror 구성은 잠시 기다리고 [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-78.png)

18. 저장소 구성은 아래 화면과 같이 선택하고 [Done] 버튼 을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-79.png)

19. 저장소 구성은 아래 화면과 같이 선택하고 [Done] 버튼 을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-80.png)

20. 계속 진행을 위한 확인 메시지 안내 화면이 출력됩니다. 이때 [ Continue ] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-81.png)

21. 사용자 계정 생성을 위해 kosa 입력하고, Your servers name : 은 kosaserver을 입력합니다. Pick your name : kosa를 입력하고 비밀번호을 입력합니다 
예) 비밀번호을 kosa1004$으로 입력하고 엔터를 눌러 다음으로 진행합니다 
![alt text](image-82.png)

22. Ubuntu pro로 업그레이드는 하지 않습니다.  [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-83.png)

23. SSH 구성은 아래 화면과 같이 선택하고
  [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-84.png)

24. 추가서버 구성은 아래 화면과 같이 선택하지 않고
  [Done] 버튼을 엔터키로 눌러 다음으로 진행합니다 
![alt text](image-85.png)


26. 설치를 진행 합니다.
![alt text](image-86.png)
설치 진행중 화면 

![alt text](image-87.png)
설치 완료 화면 

27. 설치가 완료되면 [ Reboot Now ] 버튼을 엔터키로 눌러 다시 시작합니다 
kosa 계정으로 로그인 합니다

28. 처음 리부팅을 하면 설치시 사용한 cdrom 제거을 위한 Enter key 를 눌러야 진행 됩니다 
![alt text](image-88.png)

29. 부팅완료되면 로그그인 화면이 아래과 같이 나오면 kosa 계정으로 로그인 합니다 
![alt text](image-90.png)

30. 최초 로그인을 후 시스템을 최신 상태로 업데이트 합니다 
![alt text](image-93.png)
시스템을 최신 상태로 업데이트

```
sudo apt update && sudo apt upgrade -y
```

31. 시스템을 최신 상태로 업데이트 진행 중 및 완료 화면
